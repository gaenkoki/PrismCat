# 风格与约定

- 文件编码使用 UTF-8。
- 后端以 Go 为主，结构体字段通过 `yaml` / `json` 标签与配置、API 对齐。
- 前端以函数组件 + Hooks 为主，`Settings.tsx` 一类页面文件中会用 `useCallback` / `useMemo` 管理派生逻辑和事件。
- 导入路径大量使用 `@/` 别名。
- 命名上以英文标识符为主，注释与界面文案可混合中文。
- 修改应尽量做最小化编辑，避免无关重构；该仓库当前协作偏好非常强调小 diff。
- UI 文案来自 i18n 资源文件，增删设置项时需要同步检查 `web/src/locales/*.json`。
